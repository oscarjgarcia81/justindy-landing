# Drone Soft-Wash Competitor Monitoring System

## Overview
Automated daily intelligence reports for drone-based exterior cleaning industry in the Midwest/Chicago region.

## Report Schedule
- **Daily Reports:** 9:00 AM CT (Tuesday-Sunday)
- **Monday Reports:** Extended weekend coverage (Friday 9 AM - Monday 9 AM)
- **Format:** Email with structured sections

## Monitored Elements

### Per Competitor
- [ ] News mentions and press releases
- [ ] Social media activity (LinkedIn, Facebook, Instagram)
- [ ] Website updates and changes
- [ ] Job postings (hiring = growth indicator)
- [ ] New service offerings
- [ ] Pricing changes
- [ ] Client wins/case studies
- [ ] Leadership changes

### Industry-Wide
- [ ] New drone cleaning technology announcements
- [ ] Regulatory changes (FAA, local ordinances)
- [ ] Industry partnership announcements
- [ ] Market expansion news
- [ ] Equipment/technology vendor updates
